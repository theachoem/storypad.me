## What's Changed

Improve IOS support + bug fixes + overall performance 🪴

* Close #356 refine for IOS release by @theachoem in https://github.com/theachoem/storypad/pull/357
* Fix feeling does not save
* Use Dart define to choose design system/icons at compile time instead of via context for performance.
* Add const where needed for performance
* Pass the object to db listeners directly to avoid re-fetch from db which improves performance
