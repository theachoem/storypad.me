**What’s new:**

1. Improved emoji system\
   You can now use emojis directly from your keyboard. Previously limited to 45 emojis, it’s now unlimited. We’ve also added weather and activity emojis for more expressive entries.
2. Improved tag experience\
   Tags are now added directly inside the story page. Tap to open a quick overlay, type your tag, and get instant suggestions. If it already exists, you can select it. If not, you can create it right there; no need to open the tag drawer anymore.
3. Added cursor left/right buttons (easier editing, esp. with images)
4. Added tap Home again to scroll to top (tablet UI only)
5. Added photo capture in albums
6. Added voice recording directly from home

📌 _No screenshots for this one; we’ll share more details on each feature separately later._

In the meantime, please update to the latest version to try everything out 🙌
