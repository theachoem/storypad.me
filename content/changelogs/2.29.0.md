## What's Changed

### ✨ Features
* Improved emoji system 💪 by @theachoem in #636
   You can now use emojis directly from your keyboard. Previously limited to 45 emojis, it’s now unlimited. We’ve also added weather and activity emojis for more expression.
* Improved tag experience 🏷️ by @theachoem
  Tags are now added directly inside the story page. Tap to open a quick overlay, type your tag, and get instant suggestions. If it already exists, you can select it. If not, you can create it right there—no need to open the tag drawer anymore.
* Added cursor left/right buttons (easier editing, esp. with images) by @theachoem in #633
* Added tap Home again to scroll to top (tablet UI only) @theachoem
* Added photo capture in albums @theachoem
* Added voice recording directly from home  @theachoem

### 🐛 Fixes

* Fix string not well-formed UTF-16 by @theachoem
* Only show social link when URL exists by @theachoem
