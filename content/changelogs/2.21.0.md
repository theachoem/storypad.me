## What's Changed

### ✨ New Features
- [#530](https://github.com/theachoem/storypad/pull/530) - **Dedicated iPad/tablet UI** - Enhanced experience on larger screens 📌
- [#538](https://github.com/theachoem/storypad/pull/538) - **Allow drag to switch day in calendar view** - Swipe or drag through calendar dates seamlessly
- [#537](https://github.com/theachoem/storypad/pull/537) - **Keyboard shortcuts** - Press Enter to edit story & Ctrl+S to save
- [#536](https://github.com/theachoem/storypad/pull/536) - **Named routes (Navigator 2.0)** - Improved navigation architecture
- [#535](https://github.com/theachoem/storypad/pull/535) - **Save story as template** - Reuse your favorite story formats
- [#533](https://github.com/theachoem/storypad/pull/533) - **Keyboard input support in PIN code view** - Better accessibility
- [#529](https://github.com/theachoem/storypad/pull/529) - **Add search to side menu** - Quick search accessibility
- Add take photo button to toolbar
- Include events when searching for stories
- Auto scroll to edited stories and reorder them globally

### 🎨 UI/UX Improvements

- [#531](https://github.com/theachoem/storypad/pull/531) - **Polish tablet view** - Refined tablet experience
- Use SharedAxisPageTransitionsBuilder for desktop - Quicker transitions on large screens
- Make SpTapEffect keyboard behavior consistent with inkwell
- Improve setting showtime/dayago in home view UX
- Reorder time label to leading position
- Add timeline divider to yearly view

### 🔧 Enhancements

- [#527](https://github.com/theachoem/storypad/pull/527) - **Improved yearly backup system** - More reliable data backup
- [#526](https://github.com/theachoem/storypad/pull/526) - **Upgrade to Flutter 3.38.1** - Latest framework features & performance
- [#524](https://github.com/theachoem/storypad/pull/524) - **Year-based backup** - Better data organization 📌
- Show better error UI for show/backup service
- Refactor search view to limit results on empty for better performance
- Only find IAP provider when mounted
- Add sidebar route analytics
- A/B testing setup for relax sound free set

### 🐛 Bug Fixes

- [#532](https://github.com/theachoem/storypad/pull/532) - **Fix search indexing glitches**
- Fix period calendar break style on iPad
- Fix search view keyboard glitch caused by Scrollable.ensureVisible
- Remove tap outside to dismiss keyboard (use default behavior)
- Clear snack bar before removing asset
- Remove duplicated Feedback.forTap(context)
