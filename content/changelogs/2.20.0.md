Introducing Voice Journal Add-on! 🎤✨

We're thrilled to announce the Voice Journal add-on - a powerful new way to capture your stories and thoughts through audio recording!

**Key Features:**

- 🎤 **Voice Recording**: Record your journal entries directly from the app
- 🎙️ **Easy Capture**: Quick and intuitive voice recording interface
- 🌍 **Enhanced Localization**: Day suffix now displays correctly for English language
- 🌙 **Theme Mode Button**: New theme mode toggle in the drawer for easy dark/light switching
- 🔗 **TikTok Integration**: Added TikTok as a sharing platform
- 📚 **Documentation**: See the [Voice Journal overview](https://github.com/theachoem/storypad/blob/2.20.0/docs/features/voice-journal/overview.md) for implementation details

**What's Changed:**

- Only display day suffix for English language (#518)
- Fix broken images in docs (#517)
- Add TikTok platform (#519)
- Add theme mode button to end drawer (#520)
- Upgrade package to major version (#521)
- Add voice recording feature (#522)
