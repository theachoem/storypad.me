Major platform improvements and enhanced user experience! 🚀✨

This release brings significant behind-the-scenes upgrades that make StoryPad faster, more reliable, and easier to use across all devices.

**Key Improvements:**

- ⚡ **Flutter 3.35.1 Upgrade**: Enhanced performance and stability with the latest Flutter framework
- 🎨 **Edge-to-Edge Design**: Beautiful minimal interface with transparent navigation bars for a more immersive experience
- 🔧 **Better Code Quality**: Automated testing and quality checks ensure a more reliable app
- 💳 **Simplified Purchases**: Streamlined in-app purchase flow - no account required for basic transactions
- 📱 **Full-Screen Views**: Restored immersive full-screen page viewing experience

These improvements focus on making your daily journaling smoother and more enjoyable, with better performance and a cleaner, more modern interface.
